## Why Validation Passed But Visual QC Failed

The manifest validator (`scripts/validate-jack-puppet-pack.mjs`) checks only:
- File exists at the declared path
- File size is greater than 0 bytes

This check is necessary but not sufficient. **22 of 37 puppet layer PNGs are placeholder stubs** — they exist, they are non-zero, they pass the manifest check, but they contain no production artwork. They are likely: blank canvases, solid fills, or single-shape sketches generated by an earlier throwaway pass.

The diagnostic signature is file size vs. canvas size ratio. A real 1024×1024 RGBA illustration at 72 dpi with detailed linework and shading should produce a PNG between 200 KB and 2 MB depending on complexity. Placeholder stubs compress to 3–36 KB.

### Nano Banana MCP — Status and Setup

**Nano Banana Pro** is an image generation model accessible via [Replicate](https://replicate.com). It is referenced in the research repo `research/claude-code-video-toolkit` and in `research/claude-remotion-kickstart` (which includes a `/generate-image` slash command).

**Current status:** NOT configured. Not available in this session.

**What Nano Banana supports:**
- Text-to-image generation with detailed prompts
- Style-reference images (pass an existing image as style anchor) — critical for design consistency
- Multiple output formats (PNG, JPEG, WebP)
- Does NOT save directly to project paths — returns a URL; manual download or script required
- Resolution control: can target specific pixel dimensions

**To configure Nano Banana MCP (requires explicit approval — do not install yet):**

Step 1 — Get a Replicate API key at https://replicate.com/account/api-tokens

Step 2 — Create `.mcp.json` at the project root:
```json
{
  "mcpServers": {
    "replicate": {
      "type": "sse",
      "url": "https://mcp.replicate.com/sse"
    }
  }
}
```

Step 3 — Set the API token in the session environment:
```
export REPLICATE_API_TOKEN=your_token_here
```
Or add it to `.claude/settings.local.json` as an environment variable entry.

Step 4 — Restart Claude Code. The Replicate MCP tools will appear in the session.

Step 5 — Call Nano Banana Pro via the MCP. The model ID is:
`black-forest-labs/flux-schnell` (fast) or a Nano Banana variant — confirm exact model ID from Replicate at time of use.

**Limitations:**
- Nano Banana / Replicate does not natively save to a local Windows path
- Output is a URL — needs a download step (can be automated with `fetch` or `curl`)
- Style references improve consistency but do not guarantee exact character match
- Transparent background output requires a separate background removal step
- Each generation is non-deterministic — QC gate is mandatory

## Mandatory Visual QC Gate

No generated asset enters `assets/puppet/layers/` until it passes this checklist. Failing one item = reject and regenerate.

### QC Checklist

**Identity check (for all character layers):**
- [ ] Fur colour matches `jack_01_forward_calculating.png` (golden/amber, not brown, not orange, not white)
- [ ] Nose is black and round
- [ ] Eyebrows are dark and thick
- [ ] Style is flat 2D illustration — not photorealistic, not painterly, not anime
- [ ] Linework weight is consistent with existing reference assets
- [ ] No added accessories not in the canonical design

**Transparency check:**
- [ ] Background is fully transparent (alpha channel present)
- [ ] No white halo or fringe around the element
- [ ] No background colour bleed

**Composition check:**
- [ ] Element is centred or positioned as specified in the asset spec
- [ ] Canvas dimensions match specification (1024×1024 for body/head, 512×512 for mouth/eyes, etc.)
- [ ] Element is not clipped by canvas edge

**Size sanity check (auto-detectable):**
- [ ] PNG file size is ≥ 50 KB for 512×512 layers
- [ ] PNG file size is ≥ 150 KB for 1024×1024 layers
- [ ] PNG file size is ≥ 200 KB for 1920×1080 background

### Staging Protocol

1. Generate new asset with the correct prompt
2. Save to `assets/puppet/layers_staging/[category]/[filename].png` (new directory — not live)
3. Open in any image viewer or Photopea
4. Run QC checklist
5. If passes: move (not copy) to `assets/puppet/layers/[category]/[filename].png` (overwrites placeholder)
6. Run `node scripts/validate-jack-puppet-pack.mjs` to confirm manifest still passes
7. Rebuild PSD: `node scripts/build-jack-character-animator-psd-v1.mjs`
8. Review updated layout preview PNG
