# Jack Puppet Rebuild Plan v1

Date: 2026-05-19

## Rule

Rebuild one category at a time. Do not resume PSD assembly until the rebuilt category has passed visual QC against the canonical Jack references.

No new artwork was generated by this audit. The user will generate replacement artwork manually later.

## Current Gate

The existing validator passes structurally, but that is not production approval. Visual QC failed because the active layer folder contains crude placeholders and technically invalid transparent-layer files.

PSD assembly remains paused.

## Rebuild Order

### 1. Body / Torso / Arms / Tie

Rebuild first.

Files:

- `assets/puppet/layers/body/jack_torso_front.png`
- `assets/puppet/layers/body/jack_torso_three_quarter.png`
- `assets/puppet/layers/body/jack_arm_left_resting.png`
- `assets/puppet/layers/body/jack_arm_right_resting.png`
- `assets/puppet/layers/body/jack_arm_right_tie_fix.png`
- `assets/puppet/layers/body/jack_tie_straight.png`

Reason:

Body silhouette, suit fit, seated posture, arm positions, and tie separation define the puppet's physical rig. These are currently crude vector placeholders and would break the visual credibility of every shot.

QC requirements:

- Match canonical Jack suit proportions and adult-animation style.
- Transparent PNG layers for torso, arms, and tie.
- Tie remains separate from torso for physics.
- Arm joints include enough overlap for rigging.
- Tie-fix arm must align with torso and tie knot.
